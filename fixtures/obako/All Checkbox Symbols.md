# All Checkbox Symbols

Testing all supported checkbox symbols.

- [ ] Open task (space)
- [x] Done task (x)
- [>] Forwarded task (>)
- [+] In progress task (+)
- [d] Deferred task (d)
- [A] Archived action (A)
- [-] Cancelled task (-)
- [f] Foreground item (f)
- [N] Notice item (N)
